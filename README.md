### Tech: Typescript

# Para executar

```cmd
npm i
```

```cmd
npm run start
```

ou:

```cmd
docker build --tag rinha-ts .
```

```cmd
docker run --name rinha-ts -d -p 3000:3000 rinha-ts
```

Para ver o resultado:

```cmd
docker logs rinha-ts
```
